## Description

Proyecto final para el bootcamp: JS en el backend.
Este proyecto es la API para un blog.
Incluye:
Creación (edición, visualización -de todos o por id-, eliminación) de usuarios, que pueden ser usuarios comunes o administradores.
Creación (edición, visualización -de todos o por id-, eliminación) de posts por cada usuario.
Búsqueda y filtrado de posts.
Así como funciones de administrador, tales como: obtener listado de usuarios y eliminarlos, o el control de posts (eliminar o editar).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test
